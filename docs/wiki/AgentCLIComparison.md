# Agent CLI Karşılaştırması

**Özet:** Gemini CLI, Claude Code, Kilo Code, Codeex, Codex gibi agent CLI'ların Open Design bağlamında karşılaştırması.

**Teknolojiler:** Gemini CLI, Claude Code, Kilo Code, Codeex, Codex, MCP

**Bağlantılar:** [[OpenDesign]], [[ClaudeDesignVsOpenDesign]]

**Kaynak Video:** [Open Design — Free Local Claude Design Alternative](https://www.youtube.com/watch?v=8XcbyliBwc4) (`8XcbyliBwc4`)

---

## Open Design Desteklenen Agent CLI'lar

Open Design, ilk açılışta sistemde kurulu olan agent CLI'ları otomatik algılar ve 12 farklı adapter katmanıyla entegre eder:

- **Gemini CLI** (Google)
- **Claude Code** (Anthropic)
- **Kilo Code**
- **Codeex**
- **Codex** (OpenAI)
- ve diğerleri

## Karşılaştırma Tablosu (Video'dan)

Video, agent CLI'ların şu boyutlarda karşılaştırıldığını gösteriyor:

| Boyut | Notlar |
|---|---|
| Argo shape (prompt path) | MCP üzerinden iletişim şekli |
| Stream format | JSON / Markdown |
| Bin | CLI çalıştırılabilir |
| Open Code uyumluluk | Bazı agent'larda tam destek |

## Tavsiye

Open Design ile en geniş uyumluluk için **Claude Code** veya **Gemini CLI** tercih edilir; ancak herhangi bir MCP-uyumlu agent CLI de çalışır.
